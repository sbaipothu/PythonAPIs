1. create virtual environment
python -m vevn path C:\Project_folder\venv

2. Activate virtual environment
venv\scripts\activate

3. Install djangorestframework & db related package (mssql-django for sqlserver)
pip install djangorestframework
pip install pyodbc

4. Create django project & any other sub-projects if required
Main project: django-admin startproject DjangoRestAPI (. -> optional and indicates to create project files in current directory)
sub project: django-admin startapp app_name - created subproject for sample

5. Update db config in settings.py

6. Install django-inspectdb-refactor to create db models(Note that we've few issues with latest version of this package). We can use inspectdb to create models in single python file
python manage.py inspectdb>models.py  -> used this for sample
or python manage.py inspectdb_refacor --database=dbname --app=dbfolder name
 
7.  Add rest_framework, 'inspectdb-refactor' to settings.py under INSTALLED_APPS before creating db models in step 6

8.  Add serializers.py, update models.py if model is not autogenerated

9. import model to views.py and create viewset

10. import viewsets to urls.py of the subproject 

11. import urls of subproject to main project and register the routes
